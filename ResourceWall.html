Resource-Wall

external resources library

Clon-code of Pinterest

This project is clone-code the Pinterest which is user of this website easily register and log-in, then user can see all the open resources which is saved by other users. That means, this website can see all the external resources from all the users, and user can save the resources by press 'create resource'. If a user want to save a webpage, or site, user need to press that button, and can see all the resouces from my profile.

When user click the resource, avaiable to look up the detail of the resouce. It include description and url in the bottom of the image, and like, rating, and commnets are will be on the resource detail page. The like and rating will be automately give avarage.


this project is used only Javascript and node-js. create the each routers in js and ejs files, and used the PSQL database for putting the fake resources or examples.
