Latch

Real-time matching language exchanger

Overview

This project was a 3 member-group project, and this is the proudest project I ever worked on. In this project, we utilized all the knowledge that we learn from the Lighthouse Lab and spent a lot of efforts on deciding and planning the topic and design.

As we implement the application based on the plans simulated in Figma, conflicts emerged time to time. We always started with communication about the idea and simulate in Figma, negotiated with each other. This communication and conflict-resolving experience are as valuable as the technical skills I gained.

How this works

We used React/React-Native for the front-end, Node.js for the backend, and PostgresSQL for Database. Also, We used RapidAPI for our convinience. Furthermore, I have gained much deeper understandings in WebSocket for building a real-time-matching system.
